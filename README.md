# Globomantrics
A Vuejs project using Mongodb

Make sure you have following installed

  1. Node
  2. MongoDB
  3. Studio 3T
  4. Vuejs
 
After Successfully Cloning this respiratory, DO the following in your terminal

$cd client/

$npm run build

$npm run dev

$code .   // If you're using VSCode
